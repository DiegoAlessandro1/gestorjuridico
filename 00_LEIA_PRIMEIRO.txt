🚀 COMEÇAR EM 3 PASSOS
═══════════════════════════════════════════════════════════════════════════

1️⃣ OBTER MONGODB
   Acesse: https://mongodb.com/cloud/atlas
   Criar cluster gratuito M0
   Copiar connection string

2️⃣ CONFIGURAR .env
   Abra .env → Cole URI do MongoDB
   Salve

3️⃣ EXECUTAR
   Windows: setup.bat → python run.py
   Linux/Mac: bash setup.sh → python run.py
   
   Pronto! Acesse: http://localhost:5000
