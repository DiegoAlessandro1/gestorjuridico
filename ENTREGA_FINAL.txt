🚀 COMO COMEÇAR (10 MINUTOS)
═══════════════════════════════════════════════════════════════════════════

1️⃣ OBTER MONGODB URI (2 min)
   - Acesse mongodb.com/cloud/atlas
   - Criar cluster gratuito M0
   - Copiar connection string

2️⃣ CONFIGURAR .env (1 min)
   - Abra arquivo .env
   - Cole a URI do MongoDB
   - Salve

3️⃣ INSTALAR (3 min)
   - Windows: setup.bat
   - macOS/Linux: bash setup.sh

4️⃣ EXECUTAR (1 min)
   - python run.py

5️⃣ ACESSAR (1 min)
   - http://localhost:5000

6️⃣ CARREGAR DADOS (2 min - opcional)
   - python seed_data.py
