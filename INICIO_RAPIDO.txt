## 1️⃣ OBTER MONGODB URI (2 minutos)

▶ Acesse: https://www.mongodb.com/cloud/atlas
▶ Criar conta (gratuita)
▶ Criar cluster M0 (gratuito)
▶ Connect > Python > Copiar connection string

EXEMPLO:
mongodb+srv://seu_usuario:sua_senha@cluster0.mongodb.net/gestor_juridico?retryWrites=true&w=majority

---

## 2️⃣ CONFIGURAR ARQUIVO .env (1 minuto)

Abra o arquivo .env no editor (está na raiz do projeto) e coloque:

MONGODB_URI=mongodb+srv://SEU_USUARIO:SUA_SENHA@cluster0.mongodb.net/gestor_juridico?retryWrites=true&w=majority
MONGODB_DB_NAME=gestor_juridico
SECRET_KEY=chave_super_segura_desenvolvimento_2026
FLASK_ENV=development
FLASK_DEBUG=True
PORT=5000

## ❌ ERROS COMUNS

### Erro: "Cannot connect to MongoDB"
✅ Verifique:
  1. URI do MongoDB está em .env?
  2. Senha contém caracteres especiais? Use URL encoding
  3. IP está whitelisted no MongoDB Atlas?

### Erro: "Module not found"
✅ Verifique:
  1. Ambiente virtual está ativado?
  2. requirements.txt foi instalado?
  pip install -r requirements.txt

### Página em branco
✅ Verifique:
  1. Console do navegador (F12) tem erros?
  2. Terminal do Python tem erros?
  3. Recarregue com Ctrl+F5 (limpa cache)
